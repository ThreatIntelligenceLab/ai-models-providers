# ai-models-providers

A curated collection of **390 AI models**, with structured information about their **strengths, capabilities, and practical use cases**.

## Contents

- **`models.json`** — contains data on **390 AI models**
- **`providers.md`** — contains background information on **30+ AI providers**

The information in this repository has been sourced from:

**https://feluda.ai/ai-models**

This repository is intended as a practical reference for anyone who wants to explore the AI model landscape in a more structured way, whether for research, comparison, integration, or building applications on top of model metadata.
